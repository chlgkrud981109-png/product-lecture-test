# 게임 성향 테스트 (Game Personality Test)

이 프로젝트는 사용자의 성향을 분석하여 그에 맞는 게임 장르와 대표 게임을 추천해주는 웹 서비스입니다.

## 주요 기능
- **성향 분석**: 간단한 3가지 질문을 통해 사용자의 게임 취향을 분석합니다.
- **맞춤 추천**: 분석 결과에 따라 4가지 유형 중 하나를 도출하고 관련 게임을 추천합니다.
- **다크 모드 지원**: 사용자 환경에 맞는 라이트/다크 모드 전환 기능을 제공합니다.
- **반응형 디자인**: 모바일과 데스크탑 어디서든 쾌적하게 이용 가능합니다.

## 기술 스택
- HTML5
- CSS3 (Vanilla CSS)
- JavaScript (Vanilla JS)

## 결과 유형
1. **승부욕 넘치는 스트라이커**: 경쟁과 액션을 즐기는 타입
2. **자유로운 모험가**: 액션은 즐기되 혼자만의 속도를 중시하는 타입
3. **냉철한 전략가**: 전략과 경쟁을 즐기는 타입
4. **감성적인 스토리텔러**: 여유로운 플레이와 스토리를 중시하는 타입
